# YJCocoa

集合库

---

## YJCocoa/CocoaTouchLayer/UIKit/AutoLayout

AutoLayout主要实现NSLayoutConstraint的相关扩展和封装，它能使我们关于屏幕布局的开发越来越简单。

---

## YJCocoa/CoreServicesLayer/Foundation/Log

NSLog打印优化，Unicode自动转化为中文输出。
